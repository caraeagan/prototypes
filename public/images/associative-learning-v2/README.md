# Associative Learning V2 - Image Assets

This folder should contain 6 symbol images for the test.

## Required Files:

1. `symbol1.png` - First symbol (used in test question 1 and sentence building)
2. `symbol2.png` - Second symbol (used in test question 2 and sentence building)
3. `symbol3.png` - Third symbol (used in test question 3 and sentence building)
4. `symbol4.png` - Distractor symbol (not tested, used as distractor in questions and sentence building)
5. `symbol5.png` - Distractor symbol (not tested, used as distractor in questions and sentence building)
6. `symbol6.png` - Distractor symbol (not tested, used as distractor in questions and sentence building)

## Instructions:

1. Add your 6 PNG image files to this directory
2. Name them exactly as shown above (symbol1.png through symbol6.png)
3. Recommended image size: 400x400 pixels or larger (square format)
4. Background should be transparent or white
5. Ensure images are clear and distinct from each other

## Notes:

- Symbols 1, 2, and 3 are the ones students will learn and be tested on
- Symbols 4, 5, and 6 are used as distractors in multiple-choice questions
- The correct sentence order is: symbol1, symbol3, symbol2
